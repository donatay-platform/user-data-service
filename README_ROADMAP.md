# 🗺️ Дорожная карта проекта (Roadmap / TODO)
Текущий статус разработки системы донатов для стримеров **DonationSpace**.

---

## 🛠️ ВЫПОЛНЕНО (Текущий функционал)

### 🟢 Бэкенд (donation-app)
* [x] **Архитектурный каркас**: Реактивный стек (Spring WebFlux, R2DBC PostgreSQL, Clean Architecture, Java 21, Spring Boot 3).
* [x] **База данных**: Автоматическое развертывание схемы данных через Liquibase.
* [x] **Регистрация стримера**: Регистрация по Email и защищенному паролю (шифрование BCrypt).
* [x] **Авторизация**: Выдача JWT-токена доступа.
* [x] **Безопасность (MFA)**:
  * [x] Двухфакторная аутентификация через **Google Authenticator** (TOTP).
  * [x] Двухфакторная аутентификация через **SMS-коды** (с симуляцией СМС-шлюза в логах).
* [x] **Личный кабинет стримера**: Редактирование профиля (никнейм, аватар, баннер шапки, телефон, смена пароля).
* [x] **Версионирование сервисов**: Эндпоинт `/api/version` с форматом версии `MAJOR.RELEASE.DATE.TIME.HASH`.
* [x] **Тестирование**: Юнит-тесты и веб-слайс тесты с авто-проверкой покрытия кода плагином JaCoCo (покрытие > 70%).
* [x] **Интеграция с Git**: Раздельные репозитории для бэкенда (`donatay-labudi-labuday`) и фронтенда (`donatay-client`).

### 🟢 Фронтенд (donation-client)
* [x] **Интерфейс (UI)**: Премиальный Dark-дизайн с эффектом Glassmorphic (стеклянные карточки, неоновое свечение, иконки Lucide).
* [x] **Динамические экраны**: Переключение между Регистрацией, Входом, Вводом 2FA-кода и Личным кабинетом без перезагрузки страницы.
* [x] **Превью профиля**: Визуальное отображение аватара и баннера стримера на лету при вводе URL-ссылок.
* [x] **Полная интеграция с API**: Корректная отправка CORS-запросов на `http://localhost:8080`.
